Descrição

Neste curso iremos abordar temas para iniciantes em desenvolvimento de aplicativos como, por onde começar, com o que se preocupar, o que vai aprender e muito mais.

Começaremos conhecendo a estrutura do ionic.

Depois iremos criar nossa primeira página e adicionar conteúdo a ela.

Mais adiante iremos adicionar mais páginas e até fazer uso da câmera para armazenar imagens no celular.

Iremos ver todo o caminho básico rumo a sua nova carreira de criador de apps.

O que você aprenderá

  • Desenvolver aplicativos híbridos para Android e IOS

Há algum requisito ou pré-requisito para o curso?

  • Conhecimentos básicos de Html
  • Conhecimentos básicos de Css
  • Conhecimentos básicos javascript

Para quem é este curso:

  • Qualquer um que queira aprender a desenvolver aplicativos e tenha disposição para aprender novas coisas.
